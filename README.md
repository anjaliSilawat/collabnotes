# ✏️ CollabNotes

A real-time collaborative note-taking web application where multiple users can edit the same note simultaneously — like a lightweight Google Docs.

🌐 Live Demo: https://collabnotes-liard.vercel.app

## 🚀 What is CollabNotes?

CollabNotes allows multiple users to write in the same note at the same time, see changes instantly, and save everything in real time using WebSockets.

---

## ✨ Features

- 🔐 Authentication (JWT login/register)
- ⚡ Real-time collaborative editing
- 🏠 Room-based system with shareable code
- 👥 Live user presence indicator
- 💾 Save notes to database
- 🖼️ Image upload support
- 📄 PDF/file upload support
- 🌙 Dark mode support
- 📱 Fully responsive UI

---

## 🛠️ Tech Stack

- Frontend: React.js, Vite, SCSS  
- Backend: Node.js, Express.js  
- Database: MongoDB Atlas + Mongoose  
- Real-time: Socket.io  
- Auth: JWT  
- Deployment: Vercel + Render  

---

## ⚡ How it works

Users join a room → connect via WebSockets → any text change is broadcast instantly to all users in the same room.
